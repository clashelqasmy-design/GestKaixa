En aquesta part del projecte he hagut de configurar diferents metodes per assolir lo que em demanen, que es el menú d'administrador que pugui registrar nous usuaris, obrir nous comptes, assignar un usuari a un compte i llistar usuaris i els seus comptes.

Explicacio dels diferents metodes ficats al codi:

**RegistrarNouUsuari** Demana a l'administrador les dades del nou usuari (dni, nom, cognom, adreça, telèfon, username i password) i fa un INSERT INTO Usuaris amb totes les dades introduïdes.

**ObrirNouCompte** Demana el número de compte i fa un INSERT INTO Comptes per crear-lo. L'estat i la data de creació s'assignen automàticament per la base de dades amb els valors per defecte.

**AssignarUsuariACompte** Demana el username, el número de compte i el rol (TITULAR/AUTORITZAT). Primer fa un SELECT id a Usuaris per obtenir l'id de l'usuari, després un altre SELECT id a Comptes per obtenir l'id del compte. Si tots dos existeixen, fa un INSERT INTO UsuarisComptes per enllaçar-los.

**LlistarUsuarisIComptes** Fa un SELECT amb dos INNER JOIN entre Usuaris, UsuarisComptes i `Comptes` per mostrar tots els usuaris amb els seus comptes associats i el rol de cadascun, ordenats per username.

**GestionarMenuAdmin** Bucle do/while que mostra el menú d'administrador i executa el mètode corresponent segons l'opció. No surt fins que l'administrador tria l'opció 5.
